# Papilledema Risk Calculator

A lightweight web-based demo calculator for estimating the predicted
probability of papilledema from selected clinical variables.

This repository hosts a static HTML/JavaScript implementation intended
for research, education, and collaborator demonstration. It is not
intended for standalone clinical diagnosis or treatment decisions.

## Live Demo

https://weichunlin.github.io/risk_calculator/

## Model Information

-   Model type: Multivariable logistic regression
-   Outcome: Predicted probability of papilledema
-   Implementation: Static HTML/CSS/JavaScript
-   Deployment: GitHub Pages
-   Version: v1.0.0
